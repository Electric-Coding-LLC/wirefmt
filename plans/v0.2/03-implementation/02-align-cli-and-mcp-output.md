# Align CLI And MCP Output

[Back to Execution Map](../EXECMAP.md)

## Goal

Reduce product split between CLI and MCP by aligning result semantics and shared
contracts where each interface allows.

## Tasks

- Review current CLI output and MCP result shapes side by side.
- Normalize naming and field semantics where practical.
- Keep interface-specific details only when necessary.

## Exit Criteria

- CLI and MCP no longer feel like separate products.
- Shared concepts such as warnings, findings, and issue codes are documented
  consistently.
