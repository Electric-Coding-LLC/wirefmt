# Set Up Project Foundation

## Goal

Establish the Bun and TypeScript project structure for CLI-first development.

## Tasks

- Initialize the project with Bun and TypeScript.
- Add project metadata and executable entrypoint configuration.
- Define folder structure for core logic, CLI, MCP adapter, and tests.
- Add Bun-based test tooling and supporting lint or format tooling.
- Keep files near the project target size limit where practical.

## Exit Criteria

- `wirefmt` can run locally in a minimal form.
- The repo layout supports small, focused modules.
- Bun-based tooling is ready for iterative implementation.
